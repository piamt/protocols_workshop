  • EditDistance

    Clase de utilidad que permite calcular la distancia de Levenshtein (número de sustituciones/modificaciones entre dos arrays o strings. La distancia de Levenshtein es una medida de comparación aproximada que tiene en cuenta el número de modificaciones, elementos añadidos o eliminados que se requieren para hacer que dos colecciones de elementos sean iguales. Por lo tanto, esta medida de distancia es apropiada para comparar Strings similares pero no exactamente iguales (para tener en cuenta errores tipográficos, de ortografía, etc).
